Autoclave — Heater Controller

Duration: Mar 2024 – Apr 2024

Repo: github.com/tirth2812/Heater_controller

Overview

This was a solo project where I built a heater controller using the Nuvoton Mini51 microcontroller. The system allows the user to set a temperature and timer, then controls the heater through a relay using sensor feedback to maintain the desired temperature within ±1 °C calibration and accuracy.

Features

  • User can set the target temperature and timer through a keypad.
  • Relay-based heater control with safety logic to avoid overheating.
  • 3-digit 7-segment display driven by 74HC595 shift registers shows temperature and status.
  • Settings stored in AT24C04 (I²C) EEPROM so they remain after power off.
  • Includes software PWM indicators, open/closed loop detection, and time tracking with memory save logic.

Hardware

  • MCU: Nuvoton Mini51 (Mini51ZDE).
  • Peripherals: 3-digit 7-segment display, 74HC595 shift register, AT24C04 EEPROM (I²C), analog temperature sensor (ADC), relay driver, 9-key input, regulated 5V/3.3V power supply.

Work Done

  • Programmed the Mini51 in C to read sensor data, update the display, and control the relay for stable heater operation.
  • Implemented calibration routines to achieve ±1 °C precision in temperature measurement and control.
  • Added EEPROM storage so user preferences (temperature and time) are saved even after restart.
  • Integrated I²C for EEPROM, ADC for temperature readings, and relay switching logic for reliable heater control.
  • Assembled and tested the complete system independently, ensuring consistent performance and safe operation.
