Pulse Generator

Duration: May 2024 – Jun 2024

Repo: github.com/tirth2812/Pulse_generator

Overview

This was a solo project where I built an embedded pulse generator using the Nuvoton M0518 microcontroller. The system allows the user to select and control pulse speeds (RPM) for three independent outputs through a keypad and LCD menu. It uses a state machine for menu navigation, generates precise pulse signals using timers, and stores RPM values in EEPROM for persistence.

Features

  • Set and control RPM for three independent pulse channels (P, M, and K).
  • Persistent storage of RPM values in AT24C04 (I²C) EEPROM.
  • Alternating pulse signals generated on GPIO outputs using Timer1 interrupt.
  • Character LCD interface for displaying menus and RPM values.
  • 4-key user interface (Esc, Up, Down, Enter) for navigation and input.
  • LED indicators showing active channels and output state.
  • This project/module is widely used in testing purposes and in the embroidery industry.

Hardware

  • MCU: Nuvoton M0518 Series.
  • Peripherals: Character LCD (GPIO), AT24C04 EEPROM (I²C), 4 physical keys, 3 status LEDs, relay-driven outputs on OP3 and OP4.
  • Timer: Timer1 used for square-wave pulse generation.

Work Done

  • Programmed the M0518 in C to implement a state machine-based menu system for selecting channels, setting RPM values, and activating outputs.
  • Implemented EEPROM routines for saving and retrieving RPM values to ensure persistence after restart.
  • Built the keypad input handler to allow digit-by-digit RPM entry and menu navigation.
  • Assembled and tested the complete system independently, validating functionality with LEDs, LCD feedback, and oscilloscope pulse measurements.
